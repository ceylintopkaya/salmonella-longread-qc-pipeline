📥 Data Access and Setup
Please follow the steps below to prepare the sequencing data for analysis:

1. Download the Data
The dataset is hosted on Google Drive due to file size limitations. Access the required file via the link below:
👉 Google Drive: Salmonella Raw Dataset https://drive.google.com/file/d/1K4bahHS-xQVv7I6yHQFA-zeI3tCS_c16/view?usp=sharing

2. Manual Setup
Download: Download the salmonella.fastq.gz (or the zip content) to your local machine.

Extract: If the file is zipped, extract it to obtain the salmonella.fastq file.

Organize: Move the .fastq file into the data/ directory within the project root.
